  • title esta etique nos ayudara con seo que basicamente es el titulo que tendra nuestra pagina web, pero a la vez esta ayudara a que cuando se realice una busqueda similar el usuario interesado pueda encontranos mas facil

adicional a estas etiquetas encontraremos la etique link que nos ayudara a enlazar nuestro archivo html con los otros archivos css y js y arvhios externos a los de mi carpeta, tambien estara la etiqueta author o keywords que igual que title serviran para ayudar al seo y que el usuario pueda encontranos

como puedes notar la estructura es muy clara y facil de entender

ahora bien te expicare en que momento se usa cada elemento de seccion

main

usado para contenido exclusivo que se usara en esta pagina web, se usa siempre dentro del body y sera usado una sola vez

article

usado para bloques de contenido que no necesariamente tiene que tener sentido con toda la pagina web por ejemplo una sola publicacion de blog

Section

muy similar a article, este se usa para agrupar toda la informacion de un solo tipo en el sitio web cada seccion se recomienda iniciarla con un encabezado

aside

contiene contenido que no esta directamente relacionado con el contenido principal pero puede ayudar a proporcionar informacion adicional por ejemplo un glosario, biografias o enlaces que puedan interesar
